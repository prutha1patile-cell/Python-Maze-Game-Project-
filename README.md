## Python Maze Game

A maze adventure game built with Python and Pygame where players navigate through challenging levels while avoiding obstacles and finding the exit. This project demonstrates object-oriented programming, game development fundamentals, collision detection, player movement, and event handling.

## Features

Smooth player movement and controls
Maze-based gameplay
Collision detection
Goal/exit system
Built with Python and Pygame
Great project for learning game development concepts

## Technologies Used

Python
Pygame

## What I Learned

Object-Oriented Programming (OOP)
Game loops and event handling
Collision detection
Working with sprites and graphics
Debugging and problem-solving
